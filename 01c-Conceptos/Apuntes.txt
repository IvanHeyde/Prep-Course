todo lo que escribimos en JS 'hace algo', 'retorna algo' o una combinacion de ambas. En programacion esto de clasifica como:

expressions = expresiones, 'retorna algo'
statements = declaraciones, 'hace algo'

```js
// retorna algo
1 + 1
Math.pow(2, 3) + 4;
'hola' + ' soy una expression';
```

```js
// hace algo
if (condicion) {
  // código ejecutado si es true
} else {
  // código ejecutado si es false
}
```
